Running your programs

In this section, you will learn more about running AspectJ programs in the workbench.

  1. In the Package Explorer view, find helloworld.Hello.java and double-click it to open it in an editor.
  2. Right click on Hello.java in the Package Explorer and select Java Application from the cascading Run menu. This will launch the selected class as a local Java application. Note that unless you need to run a main method that is in an aspect or use an aspectpath you do not need to use the Run > AspectJ/Java Application option.
