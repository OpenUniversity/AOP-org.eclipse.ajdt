Using the inplace Cross References view

To bring up the inplace Cross References view select the element for which you want to show the cross references in the editor and either:

  • Press Ctrl+Alt+X or
  • Select Navigate > Open Cross References

To resize the inplace Cross Reference view either

  • Grab an edge of the inplace Cross References view and resize or
  • Select Resize in the drop down menu on the inplace Cross References view and resize when prompted

To move the inplace Cross Reference view

  • Select Move in the drop down menu on the inplace Cross References view and move when prompted

To persist the position and size of the inplace Cross Reference view

  • Select Remember size and location in the drop down menu on the inplace Cross References view

To show/hide the Cross References for an entire file

  1. Press Ctrl+Alt+X to show the Cross References for the current selection in the editor
  2. Press Ctrl+Alt+X again to show the Cross References for the entire file
  3. Press Ctrl+Alt+X a third time to return to showing the Cross References for the current selection in the editor

To navigate from the inplace Cross References view to the corresponding place in the editor either

  • Select the required entry in the inplace Cross References view and press Return on your keyboard or
  • Double click on the required entry in the inplace Cross References view

To text filter the contents of the inplace Cross References view start typing in the text filter bar at the top of the inplace Cross References view and the contents of the view will update to only show contents which start with the letters you've typed.
