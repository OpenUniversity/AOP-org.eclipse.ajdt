Showing Cross References for an entire file

To show/hide the Cross References for an entire file either

  • Toggle the Show the Cross References for the entire file button in the Cross References view toolbar or
  • Check/uncheck the Show the Cross References for the entire file entry in the drop down menu in the Cross References view toolbar
