Working with aspect libraries

An aspect library is a jar file containing aspects which others can use to weave into their code. Working with aspect libraries comes in two parts:

  • Creating an aspect library
  • Using an aspect library

To create an aspect library you need to set your aspect library project to use the outjar capability. To setup your AspectJ project to use the outjar:

  1. Select the AspectJ project in the Package Explorer
  2. Right click and select Properties
  3. Select AspectJ in the list on the left of the properties page
  4. Enter a name for your aspect library in the text field next to Output jar
  5. Click Ok to close the project properties dialog
A build is triggered and the outjar you specified is created.

To use an aspect library, or in other words, to weave advice from aspects contained in an aspect library or bin directory with the classes and aspects in your AspectJ project, you need to use the aspectpath capability.

To setup your AspectJ project to use the aspectpath:

  1. Select the AspectJ project in the Package Explorer
  2. Right click and select Properties
  3. Select AspectJ Aspect Path in the list on the left of the properties page
  4. Click the Libraries and Folders tab
  5. Add the necessary jar file or bin directory using the Add XXX buttons
  6. Click Ok to close the project properties dialog
