Switching build configurations

There are a three ways to switch build configurations:

  1. Right click on the .ajproperties file in the package explorer then select Activate Build Configuration.

or

  1. Select the project for which you want to switch the build configuration.
  2. From the drop-down menu on the AspectJ Build button in the workbench toolbar, select the configuration you want to activate.

or

  1. Select the project for which you want to switch the build configuration.
  2. From the workbench menubar select Project > Active Build Configuration, then select the configuration you want to activate.
