Saving a build configuration

To save a build configuration:

  1. Select the project for which you want to save the build configuration.
  2. From the menu bar, select AspectJ Tools > Save Build Configuration As....
  3. Optionally change the name of the build configuration file.
  4. Click OK to complete. A new build configuration file is created with a .ajproperties extension.
