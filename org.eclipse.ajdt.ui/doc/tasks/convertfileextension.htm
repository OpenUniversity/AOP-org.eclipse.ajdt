Converting the extensions for individual files

To convert one or more .java files to have a .aj file extension, select the files in the package explorer. Right click then select AspectJ Tools > Convert File Extension to .aj.

To convert one or more .aj files to have a .java file extension, select the files in the package explorer. Right click then select AspectJ Tools > Convert File Extension to .java.
