Opening a type in the type hierarchy view

There are several ways to open a type hierarchy on an Aspect or Java type:

  • Press F4 after selecting the type.
  • Drag and drop the element to the Hierarchy view.
  • Press Alt+Shift+H and select a type from the list in the resulting dialog.
  • Select Navigate > Open Java Type in Hierarchy from the menu bar.
  • Select Focus On... from the pop-up menu of the type hierarchy viewer.
